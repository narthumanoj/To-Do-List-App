# ✅ To-Do List App (Command Line)

A simple Python terminal-based to-do list app. You can add, view, and mark tasks as done.

---

## 📋 Features

- Add new tasks
- View all tasks with status
- Mark tasks as completed
- Exit app anytime

---

## 🔧 Technologies Used

- Python 3

---

## ▶️ How to Run

Make sure Python is installed.

In terminal:

```bash
python todo.py
